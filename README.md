# Foundary AI

A modern, comprehensive platform for technical and non-technical interviews. Foundary AI features a live coding environment, resume intelligence, knowledge graph analytics, voice interviews, and a dedicated recruiter workspace. Built with React, TypeScript, and Vite.

## Features

- **Dashboard**: High-level overview of interview metrics and candidate evaluations.
- **Interview Studio**: Complete setup for managing and conducting interviews.
- **Coding Environment**: Live, interactive coding environment for technical assessments.
- **Resume Intelligence**: AI-powered analysis of candidate resumes.
- **Knowledge Graph**: Advanced candidate skill mapping and visual analytics.
- **Voice Interview**: Audio-based interview module.
- **Recruiter Workspace**: Dedicated tools for recruiters to manage candidates and schedules.
- **Analytics**: Deep dive into hiring pipelines and performance.

## Tech Stack

- **Frontend Framework**: React 19
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling/UI**: Radix UI / Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To build the app for production, run:

```bash
npm run build
```
